# Welcome to Our Design System

This is an introduction to our component library and design system.

## Key Principles

- **Consistency**: All components follow a unified design language.
- **Accessibility**: We prioritize inclusive design in all our components.
- **Flexibility**: Components are designed to be adaptable to various use cases.

## Getting Started

To use our design system in your project, follow these steps:

1. Install the package: `npm install our-design-system`
2. Import components: `import { Button } from 'our-design-system'`
3. Use in your React application: `<Button>Click me</Button>`

## Available Components

- Buttons
- Input fields
- Modals
- Navigation bars

For detailed documentation on each component, please refer to their individual pages in the sidebar.
